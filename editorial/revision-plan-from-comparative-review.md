# Revision Plan from Comparative Review

This plan translates `editorial/christian-bestseller-comparative-review.md` into manuscript tasks.

## P0 — necessary before pastor review

- [ ] Expand `04-singleness-and-dignity.md` into a real theology of singleness: Christ, new covenant family, church belonging, vocation, desire, grief, and dignity.
- [ ] Add a chapter after chapter 04: `05-why-marry.md` — “Не лише кого знайти, а навіщо будувати шлюб”.
- [ ] Expand boundary/safety content into a full chapter: normal difficulty vs immaturity vs sin vs danger vs different format.
- [ ] Add a repeated authority label to each chapter: direct biblical teaching / biblical principle / pastoral application / practical advice.

## P1 — needed for readability and usefulness

- [ ] Add composite pastoral examples to each major part.
- [ ] Add a stage discernment map appendix: before dating, early interest, intentional dating, serious discernment, engagement.
- [ ] Add pastor/mentor do/don’t appendix.
- [ ] Add end-of-chapter practice: prayer, Scripture, counsel, observation, one concrete step.

## P2 — useful polish

- [ ] Add a short appendix comparing tools under Scripture: typology, love languages, boundaries, personality language, pastoral counsel.
- [ ] Add a “memorable phrases” pass so the manuscript repeats a few strong book-level lines.
- [ ] Add a source note page naming comparable books as influence/context, not authorities.

## Guardrails while revising

- Do not make the book a dating pipeline.
- Do not make marriage the hidden goal for every reader.
- Do not make compatibility central in the first third of the book.
- Do not quote or paraphrase comparable books closely; use the lessons structurally.
- Keep typology limited, late, and under Scripture.
