# Compatibility for Single Christians

**Ukrainian title:** «Сумісність для одиноких християн»

This repository contains the clean book manuscript derived from the Before We Build biblical-hermeneutical research layer.

The book is written in Ukrainian for single Christians, pastors, mentors, and churches. Its center is Scripture, not typology. Compatibility is treated as a practical wisdom category that must remain under Christ, Scripture, love, holiness, character, faithfulness, prayer, observation, and wise church counsel.

## Repository structure

```text
manuscript/   Clean chapter draft of the book
appendices/   Reader-facing appendices
editorial/    Outline, thesis, style guide, and review checklist
references/   Public-safe links back to the research basis
```

## Boundary

This is the **book repository**, not the typology research repository.

- Research, hermeneutical notes, claims register, risk register, and typology-integration notes remain in `before-we-build-research/biblical-compatibility/`.
- This repository keeps only the reader-facing manuscript and editorial checks derived from that research.
- Typology is not presented as biblical truth, spiritual authority, marriage guarantee, or destiny logic.
